## Backend  
Requirements:  
- .NET SDK 9.0
- Docker
- Terminal/Command Prompt  
- Visual Studio Code (optional)  

Run the application:  
```bash
docker build -t backend .
docker run -p 8080:8080 backend
```
Site will be available on http://http://localhost:5253/api
